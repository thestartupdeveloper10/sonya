Digital Portfolio Project

Unit 3 Work 2025

Project Overview

Development of a comprehensive digital portfolio showcasing my work and skills as part of Unit 3. This project involved designing, creating, and testing a professional portfolio website that demonstrates my capabilities to potential employers.

Learning Aims

Project Objectives

  • Design a digital portfolio
  • Create & test a digital portfolio
  • Review the digital portfolio

Technologies Used

  • HTML5 & CSS3
  • JavaScript
  • Web Design Tools

Project Documentation

Download Design Plan Download Project Plan

Development Process

Stage 1: Design Phase

  • Created storyboards and structure charts
  • Planned assets and software requirements
  • Designed page layouts and navigation
  • Established color schemes and typography

Stage 2: Development Phase

  • Created web pages including:
    • Home page with introduction and section links
    • Section pages with purpose descriptions
    • Context pages for content items
  • Implemented interface elements:
    • Color scheme and font formatting
    • Navigation bar and internal links
    • Images and media assets

Stage 3: Testing Phase

  • Gathered and created portfolio content
  • Converted files to appropriate formats
  • Established logical folder structure
  • Tested functionality and usability
  • Collected and implemented user feedback
